[Основные команды]:
ПКМ + R - меню взаимодействия с игроком
/act - меню действий
/testid | /testnick | ПКМ + X - проверка на ЧС
/testfind - проверка списка сотрудников на ЧС
/settings - настройки скрипта
/blmenu - редактирование Черного списка
/ud - показать удостоверение
/sud - спросить не нужна ли помощь
/viz - передать визитку
/show - полезная информация (устав и прочее)
/editshow - редактор полезной информации
/sobes - проверка на собеседовании
/autopass - настройка AutoPass
/rn | /fn - нонРП рация организации
/checkfind - проверка на прогульщиков

[Дополнительные команды]:
/binder - внутриигровой биндер
/notepad - меню заметок
/drone - управление дроном (кам-хак)
/recon - переподключение к серверу
/nrecon - переподключение со сменой ника
/rep - написать в репорт
/relog - пере-авторизация через /mn
/hist - аналог /history, по id
/setweather - установить погоду по id
/settime - установить время
/cc - очистить чат
/checkid - узнать информацию о игроке
/aad - автоподача объявлений
/ansms - ответ последнему написавшему

[Для старшего состава]:
/dmenu - меню опечатки имущества
/uninv - уволить игрока с РП
/vig - выдача выговора
/gmenu - гос. новости (3 строки)
/gstart - гос. новости (начало)
/gcont - гос. новости (продолжение)
/gend - гос. новости (окончание)

[Переменные для биндера]:
{my_nick} - Ваш никнейм с _
{my_name} - Ваш никнейм без _
{my_id} - Ваш ID
{my_number} - Ваш номер телефона
{my_org} - Ваша организация
{my_fraction} - Ваше подразделение
{my_rang} - Ваш ранг
{my_rangname} - название Вашего ранга
{my_sex} - Ваш пол
{date} - текущая дата в формате DD.MM.YY
{time} - текущее время в формате HH:MM:SS
{b_text} - текст указанный после команды
{p_id} - id, указанный после команды
{p_idbynick} - узнает id игрока по нику
{p_nick} - узнает ник игрока по id
{p_name} - узнает РП никнейм по id
{p_org} - узнает организацию по id
